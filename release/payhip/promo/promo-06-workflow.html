Quick start

Import in four steps

Install ZXP → open panel → select file → Import

1

Install

Install MD-Importer-1.6.0.zxp and restart InDesign or Illustrator.

2

Open panel

Window → Extensions → MD Importer. Choose document target.

3

Select .md

Browse for your Markdown file. Pick Automatic or Manual mode.

4

Import

Styled text frame ready — edit styles, layout, export.
