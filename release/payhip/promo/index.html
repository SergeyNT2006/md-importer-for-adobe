Откройте нужный слайд в браузере при масштабе 100% и сохраните как PNG (скриншот блока 1200×675 или 1200×1200).

Как получить PNG

  • Chrome / Edge: F12 → выделите .canvas → правый клик → Capture node screenshot.
  • Windows: Win+Shift+S — обрежьте по рамке слайда (масштаб браузера 100%).
  • Квадратная обложка: promo-07-square.html (1200×1200).
