MD Importer — Markdown Importer for Adobe InDesign and Adobe Illustrator

Minimum settings. Maximum speed.

Turn your Markdown files into styled Adobe documents in seconds. MD Importer reads .md, .markdown, and .txt files and places structured text into InDesign or Illustrator with paragraph and character styles — headings, body text, lists, emphasis, code, tables, links, and more.

No copy-paste marathon. No manual style tagging line by line. Choose a file, click Import, and start editing in Adobe with real document styles.

2. Open Adobe InDesign or Illustrator (with a document open).

4. Choose a Markdown file → Import.

KEY FEATURES

CEP panel (Window → Extensions → MD Importer)

- Automatic mode — creates and applies styles from Markdown roles (H1–H6, Body, lists, code, blockquotes, and more)
- Manual mode — map Markdown roles to your existing document styles via an editable table
- Document target — import to the active document, open an existing file, or create a new document
- Style prefix — custom prefix for generated styles (default MD, e.g. MD-H1, MD-Body)
- Serif / Sans Serif presets — base fonts for headings and body text
- Silent mode — import without completion alerts (ideal for batch work)
- Colored mode (InDesign only) — assigns distinct MD-Color-* swatches so you can see document structure at a glance before final typography

Supported Markdown

Headings, bold/italic/code/strikethrough, bullet and numbered lists, task lists, blockquotes, fenced code blocks, horizontal rules, GFM tables, links, images, footnotes, and controlled inline HTML (sup, sub).


INDESIGN VS ILLUSTRATOR

Adobe InDesign is the primary target for production work:

- Native Table objects from GFM tables
- Real footnotes from Markdown [^id] syntax
- Linked images when the file path resolves
- Colored proofing mode with MD-Color swatches

Adobe Illustrator is supported with a stability-first import path — great for quick styled text placement and typography prep. Tables, footnotes, and images behave differently than in InDesign; see the included manual for details.
