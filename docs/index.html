Import Markdown into Adobe InDesign and Adobe Illustrator with paragraph and character style mapping.

Features

  • CEP panel — Automatic & Manual import, document target (active / open / new)
  • Manual style mapping table — Role, style name, font, size
  • Style prefix (e.g. MD-H1, MD-Body)
  • Serif / Sans Serif base fonts for headings and body
  • Colored (InDesign only) — MD-Color swatches for visual markup
  • InDesign — native tables, footnotes, lists, heading spacing
  • Illustrator — import with style mapping

Free demo

Standalone ExtendScript — no install. Imports the first 80 lines of a Markdown file into InDesign or Illustrator.

Download MD-Importer-Demo.jsx

  1. Open Adobe InDesign or Illustrator (document open or create new).
  2. File → Scripts → Other Script… → select MD-Importer-Demo.jsx.
  3. Choose a .md file (see sample files) → Import.
